# AwayJS Player

Dependecy for AwayJS applications requiring scripting abstraction for different converted syntax trees such as AS2, AS3 etc.

## AwayJS Dependencies

* core
* graphics
* renderer
* scene
* stage

## Internal Structure

* adapters<br>
Adapters from AS2 classes to AwayJS

* factories<br>
Factory for AS2 SceneGraphs
